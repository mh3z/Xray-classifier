Chest X-ray anomalies classifier

This Project


This web corresponds to the result of a final university project,
which consists of making a machine learning application using the knowledge acquired.
the application consists of a deep learning system,
which classify a chest radiography, in order to detect the follow anomalies:
Covid-19. - Pneumonia. - Lung opacity. - Tuberculosis.
